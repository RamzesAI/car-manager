## Installation

```bash
$ npm install
```

## Running the app

```bash
# mongo db
$ install docker desktop
$ npm run start:mongo

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Swagger

For the convenience of testing queries, swagger is implemented
# http://localhost:3000/api

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
